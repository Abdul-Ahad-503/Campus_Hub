# CampusHub Design Configuration

## App Overview

- **App Name:** CampusHub
- **Tagline:** "Your Complete University Companion"
- **Target Users:** University students and staff
- **Platform:** Android and iOS
- **Style:** Modern, clean, minimal, student-friendly

## Design System

### Color Palette

```dart
// Primary Colors
Primary: #1E88E5 (Blue)
Secondary: #26A69A (Teal)

// Feature Colors
Lost Items: #E53935 (Red)
Found Items: #43A047 (Green)
Events/Notice: #FB8C00 (Orange)

// Base Colors
Background: #FFFFFF (White)
Surface: #F5F5F5 (Light Gray)
Text Primary: #212121 (Dark)
Text Secondary: #757575 (Gray)
```

### Typography

- **Headings:** Poppins Bold/SemiBold
- **Body:** Roboto Regular
- **Style:** Modern, readable, clean

### Design Style

- Material Design 3 inspired
- Rounded corners (12-16px border radius)
- Soft shadows
- Card-based layouts
- Bottom navigation bar
- Consistent 16px margins
- 8px grid system

### Screen Size

- **Target:** 390 x 844 pixels (iPhone 14 / Modern Android)

## Implementation Notes

This design system will be consistently applied across all features including:
- Lost & Found items
- Events & Notices
- Navigation
- Cards and components
- Forms and inputs
